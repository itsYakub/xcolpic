===============================================================

       Access an X session's pixel data from the display.

===============================================================

	Getting started:

	1. Prerequesites:
	- Git - version control;
	- Xlib - library for interaction with X11;
	- X11 running on your system;

	2. Getting the repository:

		$ git clone https://github.com/itsYakub/xcolpic.git
		$ cd xcolpic/

	3. Building the project:

		$ make all

	4. Installing the project in your system's PATH (/usr/local/bin/):

		$ sudo make install

	5. Cleaning the binaries:

		$ make clean

	6. Removing the project from your system:

		$ sudo make uninstall
